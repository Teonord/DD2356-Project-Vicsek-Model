Generates and returns a random double-\/precision float in range \mbox{[}0.\+0, 1.\+0\mbox{]}. 

Function uses the function rand() from $<$stdlib.\+h$>$ to generate a random integer value between 0 and RAND\+\_\+\+MAX. This value is then cast to a double-\/precision float and divided by RAND\+\_\+\+MAX to get a random value between 0 and 1. Values used in numerous positions in the code.
