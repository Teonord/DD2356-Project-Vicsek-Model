Main function of the simulation. 

This function performs the entire simulation loop for the Vicsek model of Flocking Birds in a serial manner. Using functions in \mbox{\hyperlink{proj__common_8h}{proj\+\_\+common.\+h}} the model is solved and results are printed in one line per timestep. The resulting values can then be pasted into the Python program to visualize the flock.
